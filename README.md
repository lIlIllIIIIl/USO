# USO - Créateur de playlist Osu → Spotify

**USO** est un outils qui permet à n'importe qui de créer des playlist spotify à partir des beatmaps les plus jouées par un joueur d'Osu.

Ce site a initialement été développé dans le cadre d'un projet scolaire en compagnie de **Jaël Beining**.

Ce projet est développé en **Vue.js** & **Node.js**

Le site est disponible sur https://uso-psi.vercel.app/
